Portafolio web personal diseñado para mostrar mis proyectos como Desarrollador Full-Stack e Ingeniero de Software. Construido como una Single Page Application (SPA), con una interfaz moderna y un sistema de diseño propio basado en elevación de colores oscuros.

---

## ✨ Características Principales

* **Arquitectura SPA:** Navegación instantánea y fluida sin recargas utilizando `react-router-dom`.
* **Diseño UI/UX:** Sistema de colores personalizado implementado con variables CSS nativas y Tailwind CSS v4.
* **Carrusel Interactivo 3D:** Visor de imágenes personalizado estilo "Cover Flow" construido desde cero con `useState`. Incluye transiciones matemáticas, soporte para desenfoque dinámico y función de Lightbox (pantalla completa interactiva).
* **Responsive Design:** Adaptación perfecta a dispositivos móviles, tablets y monitores de gran formato.
* **Scroll Suave y Animaciones:** Transiciones nativas y animaciones fluidas al navegar entre las distintas secciones de la página principal.
* **Íconos Vectoriales:** Integración de `lucide-react` para una iconografía limpia, moderna y escalable.

---

## 🛠️ Tecnologías Utilizadas

* **Librería UI:** React 18 (Hooks: `useState`, `useEffect`)
* **Build Tool:** Vite
* **Estilos:** Tailwind CSS v4
* **Enrutamiento:** React Router DOM
* **Iconografía:** Lucide React
